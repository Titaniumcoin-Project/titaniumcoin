What is Titaniumcoin?
----------------

Titaniumcoin is a hard fork of Litecoin that uses scrypt as its proof-of-work algorithm.
 - 2 minute block targets
 - subsidy halves in 300k blocks (~4 years)
 - ~300 million total coins
 - 30 coins per block
